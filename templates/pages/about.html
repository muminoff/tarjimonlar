Лойиҳа ҳақида

Лойиҳадан кўзланган мақсад — гуруҳ иштирокчилари учун Фейсбукда имкони йўқ бўлган қулайликларни сайт кўринишида дастурлаш. Сайт Фейсбук тармоғига тўғридан-тўғри бириктирилгани боис сайтдаги маълумотлар ҳар 20 дақиқада автоматик равишда янгиланиб турилади. Сайт коди очиқ.

Кредитлар

Сайтни тўлақонли ишлаши учун қуйидаги технологиялардан фойдаланилди:

  • Python дастурлаш тили
  • Django веб фреймворк
  • Facebook API хизмати
  • Twitter Bootstrap CSS фреймворк
  • Flat UI CSS дизайн пакети
  • jQuery Javascript кутубхонаси
  • D3 Javascript кутубхонаси
  • PostgreSQL маълумотлар базаси
  • ElasticSearch қидирув сервери
  • Redis тезкор маълумот структураси сервери
  • Ubuntu очиқ кодли операцион тизими
  • Amazon Web Services платформаси

Эслатма

Гуруҳ ҳақидаги барча маълумотлар Фейсбук API орқали олинади. Сайтда контентни таҳрирловчи администратор ёки модератор йўқ. Маълумотлар интернетда очиқ оммага чиқарилмайди, яъни сиз гуруҳда ёзган пост ёки шарҳ (Google, Yandex) қидирув тизимларида чиқмайди.

Сайт ҳақидаги фикр-мулоҳаза, шикоят ва таклифларингизни гуруҳда алоҳида пост очиб ёки dev@tarjimonlar.org почта манзилига ёзиб билдиришингиз мумкин.
